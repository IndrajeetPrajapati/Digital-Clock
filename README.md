# Digital Clock

A stylish and responsive digital clock built using **HTML, CSS, and JavaScript** that displays real-time hours, minutes, and seconds with a modern user interface.

## ⏰ Features

* Live Real-Time Clock
* Hours, Minutes, Seconds Display
* 12-Hour / 24-Hour Format Support
* AM / PM Indicator
* Clean and Modern UI Design
* Fully Responsive Layout
* Smooth Time Update Every Second

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 📌 Purpose

This project was created to practice JavaScript Date object, timing functions, DOM manipulation, and responsive frontend design.

## 🚀 How It Works

1. Fetches current system time.
2. Updates every second automatically.
3. Displays time in digital format.
